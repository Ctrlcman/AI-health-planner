# 🏋️‍♂️ AI 健康与健身计划助手

通过 Google Gemini 大模型，基于用户信息一键生成个性化的饮食和健身计划！  
输入年龄、身高、体重等基本信息，AI 将为你制定科学、实用、可执行的健康方案 💪

---

## 📌 项目功能

✅ 一键生成个性化 **饮食计划**  
- 自动匹配三餐 + 加餐建议  
- 结合你的健身目标和饮食偏好（如素食、生酮等）

✅ 一键生成个性化 **健身计划**  
- 包含热身、训练、拉伸环节  
- 给出动作、次数、组数，适合初学者  
- 根据性别、目标（如减脂、增肌等）精准定制

✅ 支持 **中文 AI 问答**  
- 针对生成的饮食与训练计划提问，AI 即时答复
